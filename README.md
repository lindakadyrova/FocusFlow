# Focus Flow - Android Productivity App

Focus Flow is an Android productivity app that helps users break down large tasks into manageable smaller tasks using a structured, step-by-step approach.

## Features
- Task Breakdown: Convert big tasks into smaller, actionable steps
- Task Management: Delete sub-tasks or entire big tasks  
- Progress Tracking: Mark sub-tasks as done to track completion
- Clean UI: Modern, intuitive interface with responsive design

## Tech Stack
| Tool | Purpose | 
|------|--------|
| Kotlin | Primary programming language |
| Android Studio| Development Environment |
| XML | UI layout design |
| ConstraintLayout | Responsive UI design |

## Device Compatibility
### Optimized For
- Virtual Device: Pixel 8 Pro
- Screen: 6.7 inches
- Resolution: 1344 × 2992 px

### Minimum Requirements  
- Minimum SDK: API 29 (Android 10.0 "Q")
- Target SDK: API 34 (Android 14)
- Tested On: Pixel 8 Pro emulator with recommended settings
